# University Management System

This project is a database project created for the B103 Databases & Big Data module at Gisma University of Applied Sciences. It is designed to manage university data such as students, departments, instructors, courses, enrollments, and grades.

## Project Objective
The aim of this project is to design and implement a relational database using SQL. The system demonstrates data storage, retrieval, updates, deletions, joins, aggregations, and views.

## Main Tables
- departments
- students
- instructors
- courses
- enrollments
- grades

## Features
- Relational database design
- Primary keys and foreign keys
- Constraints such as NOT NULL, UNIQUE, and CHECK
- CRUD operations
- Join queries
- Aggregate queries
- SQL views

## Files Included
- `university_management_system.sql` – full SQL script
- `DML and DQL.sql` – All the query Applied
- `ER_diagram.png` – ER diagram
- `screenshots/` – query output screenshots

## How to Run
1. Open XAMPP and start Apache and MySQL.
2. Open phpMyAdmin.
3. Import the `university_management_system.sql` file.
4. Run the SQL queries using DML and DQL.sql file to test the project.
